# Python

Objetivo:

Utilizar el archivo hecho en clase y cambiarlo para que acepte el siguiente diagrama de sintaxis:

EXP -> ( FUN PARAMS ) 

FUN -> OP | ID 

OP -> + | - | * | / | % 

PARAMS -> PARAM PARAMS | vacio

PARAM -> EXP | ID | INT | BOOL | LIST 

BOOL -> #t | #f 

LIST -> ‘( PARAMS )

Nota: ‘ se cambio a & para facilitar el reconocimiento.
